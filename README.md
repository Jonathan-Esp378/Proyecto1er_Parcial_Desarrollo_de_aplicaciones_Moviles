# Proyecto 1er parcial

En este programa creado en flutter, encontraras las practicas realizadas que se hicieron en el 1er parcial, como por ejemplo, los hola mundos, que aparezca 1 vez o 10 veces, el hola mundo, el juego de piedra papel o tijera, ademas el formulario que se hizo tambien. Aqui encontraras la nueva parte que es el kit offline, este consiste en 4 programas, 1 que es un bloc de notas rapida, despues encontraras un programa en el cual calcula tu nivel de peso, al preguntarte tu estatura y peso actual, 
despues encontraras 4 imagenes de las cuales cuando se seleccionan puedes visualizar la imagen y su titulo asignado, y por ultimo hay un juego en el que compites contra la cpu, consite en que el usuario eliga un par o impar, el sistema elige un numero al azar de lo cual se le sumara el numero que escogio el usuario, si el numero sale par y es impar o vicerversa, el usuario pierde. A continuacion se mostrarn capturas del programa funcionando.

Para poder ejecutar esta aplicacion es necesario tener flutter instalado en tu computadora, al igual necesitas instalar las extensiones de flutter en visual studio Code, cuando tengas el codigo en tu equipo presionas ctrl+shift+p y se te desplegara un menu como el siguiente:
<img width="588" height="101" alt="image" src="https://github.com/user-attachments/assets/f0e3d95c-0096-4054-9c3a-04ffe5ab586d" />
Una vez se te despliegue este menu vas a seleccionar select device, y vas a seleccionar la plataforma en la que quieres que se ejecute esta aplicacion, en mi caso la voy a ejecutar en mi navegador Edge:
<img width="585" height="103" alt="image" src="https://github.com/user-attachments/assets/54b25dc8-a9e4-482a-88e7-4311518ddc0d" />
Una vez hayamos seleccionado la plataforma vamos a ejecutar nuestro archivo main.dart, este es nuestro archivo principal, le vamos a dar en el boton llamado "Start Debugging":
<img width="63" height="64" alt="image" src="https://github.com/user-attachments/assets/9d5a5ebd-4f63-447d-81dd-c2f58bb6ee0a" />
Una vez le demos a este boton va a comenzar a iniciarse el programa, tarda aprox 1 minuto asi que ten paciencia :D
Nota: Deberas tener acceso a internet para poder ejecutar la aplicacion, de lo contrario no servira.
